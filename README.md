* Arduino UNO
* Sensor DHT11
* NodeJS
* MongoDB

##  Library NodeJS yang dipakai :
*   express
*   ejs, ejs-mate
*   mongoose

##  Keterangan file :
*   Folder arduino saya gunakan menyimpan source code arduino
*   app_monitoring, folder untuk code app nodeJS nya
